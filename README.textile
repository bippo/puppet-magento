h1. magento

This is the magento module.

h2. magento::wiz

Depends on git package:

bc. package { git: ensure => present }
include magento::wiz

Git clones *wiz* from git://github.com/bippo/wiz.git
Puts it in @/usr/share/wiz@
And creates a symlink in @/usr/bin/wiz@
